# 🏦 Bank Management System

A simple Bank Management System built with Python using Object-Oriented Programming (OOP).

## Features

- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Show All Accounts
- Save Data to File

## Technologies Used

- Python
- Object-Oriented Programming (OOP)
- File Handling

## Project Structure

```
Bank-Management-System/
│── main.py
│── accounts.txt
└── README.md
```

## How to Run

```bash
python main.py
```
